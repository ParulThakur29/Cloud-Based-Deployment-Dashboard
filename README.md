# Cloud-Based Deployment Dashboard 🚀

## 📌 Features
- Deploy applications
- Search apps 🔍
- Delete apps 🗑️
- User login system 👤

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite
- Docker

## ▶️ How to Run
1. Clone repo:
   git clone https://github.com/ParulThakur29/Cloud-Based-Deployment-Dashboard.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run app:
   python app.py

4. Open:
   http://127.0.0.1:5000
